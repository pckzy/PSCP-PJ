## How to play

เลือกความยาวตัวอักษรของคำที่ต้องการเล่น สามารถเลือกได้หลายความยาว  
กดปุ่ม Play! เพื่อเริ่มเกม  
พิมพ์ตามคำที่ตกลงมาให้ทัน พิมพ์ถูกจะได้รับแต้ม ถ้าพิมพ์ไม่ทันเลือดจะลดลงทีละหนึ่ง หากเลือดหมด --> Gameover
